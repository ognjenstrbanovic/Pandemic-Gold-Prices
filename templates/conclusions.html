Conclusions / summary / statistics / methods

We attempted to answer the question "Is gold a good hedge during a pandemic. We wondered if buying gold at the start of a pandemic would be a good idea. The question is nuanced because it doesn't ask how long you plan to keep gold for. We decided to use 2 benchmarks to analyse the question. Since there are few pandemics we found it hard to do much serious statistical analysis because of the low sample sizes. There have been only four large pandemics before COVID 19 when a series of financial data had been kept. We also found it difficult to find accurate counts of deaths in the USA in particular since these have not been recorded accurately in the past. We did find some good monthly deaths for the 1958 and 1968 flu pandemics and there is plenty of data for covid 19 which occurded in the "data age"
In order to do some statistics we had the idea that attempting to correlate market moves with the previous months deaths might be useful since we had quite a few months worth of data ( say 30 ). We decided to use the Dow Jones industrial Index as a yardstick to measure the price of gold by. since the beginning of the DOw it always been a higher number than GOLD. The ratio of DOW/GOLD has varied a great deal over the last 100 years. During the Spanish Flu it was closer to say 3. Around 2000 it was as high as 40 and recently it is around 15. If that number reached the value of One then gold would be at it's highest price relative to the DOW ever. As a general rule it is probably worth noting that generally that value has tended to increase ie stocks have outperformed gold in the long term. All the figures presented here perhaps merely reflect underlying trends not a cause and effect. Data was found in various sources : Some data we had to pay for ( $4 ! ). We first looked at the change in the DOW?GOLD based on the previous months deaths during the 1957 , 1968 and COVID pandemics. Putting all the pandemics into one data set we could find very little correlation between the deaths and the market. We attempting to draw a line of correlation but the points are far from it with many outliers. The "R squared" valuie is 0.005 which suggest no correlation at all. You would want an R squared as high as say 0.7 to make a statistical inference. We then looked at the change in the DOW from the start of each pandemic. https://github.com/pandemicgoldprices/pandemicgoldprices/blob/master/data/change_in_dow_v_gold_pandemic_outset.csv ( graph ) using this measure in the four most recent pandemics : 1958, 1969, 2009, 2020 . If you kept gold for one Year only you would have been better off but after that the benefits often dissapated. We noted the huge swing towards gold in the early 1970s after the Hong Kog flu. This we decided was most likely due to the extraordinary inflation of the period (CPI as high as 12% in 1975 ) . People purchased gold becuase their money was become worthless overnight ! This was also when many worldwide governments left the gold standard causing wild swings in relative currency prices. That uncertainty also made gold attractive.

Methods

We searched for data initially. Deciding on the NY times github repository of covid data. https://github.com/nytimes/covid-19-data We used the St louis Fred API for recent market data https://fred.stlouisfed.org/ Other sources of older data included. goldpricehistroy1850to2011 : http://www.nma.org/pdf/gold/his_gold_prices.pdf older data : https://www.measuringworth.com/datasets/gold/result.php pandemic deaths per month https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2374803/ covid deaths per month : https://www.cdc.gov/nchs/nvss/vsrr/covid19/index.htm First we compiled the older data into spreadsheets in excel and did some cleaning and graphing . These were the 3 final data sets we compiled in csv form DOW and GOLD data and the ratio for 100 years : https://github.com/pandemicgoldprices/pandemicgoldprices/blob/master/data/dow_gold_div_change_1915_2020.csv Change in the Ratio with it set to zero from the start of each pandemic. Each month of each pandemic the change in the ration is measured from pandemic outset. https://github.com/pandemicgoldprices/pandemicgoldprices/blob/master/data/change_in_dow_v_gold_pandemic_outset.csv A dataset that shows the monthly deaths for 3 pandemics ( 1958, 1969, 2020) and the change in the following month ratio. https://github.com/pandemicgoldprices/pandemicgoldprices/blob/master/data/pandemicmonthlydeaths.csv We created a mongoDB database and wrote code in python using pandas and other methods to store 2 of the datasets in mongo under different schema names. In what could be described as a "merely academic" process we decided to reload the CSV files into mongo and empty the DB and reload when the flask app is started. We note we could have just used a csv file for the whole shebang but incorporating mongo was a goal. We created a flask application in python. In the flask application we created routes for the various webpages. in the flask app when the covid19 page is called the application responds to the get request by a call to the Fred API for the last 6 months or so of data. At the front end that webpage loads CSV data from the NYTIMES github directly. Buttons allow the recent NYTIMES and FRED data to be displayed. Graphs of the monthly deaths / market moves and the change in the dow for each pandemic where created in excel. For the other pandemics the flask route requests data from mongo for a range of years which is then plotted in the front end using plotly. Webpages were styalised using BOOTCAMP and CSS.
